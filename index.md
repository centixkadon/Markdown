# Markdown Syntax Summary

## Headings

# The largest heading (h1)
## The second largest heading (h2)
###### The smallest heading (h6)

## Styling text

**bold text**

__~~bold text~~__

_italics text_

*~~italics text~~*

**_bold italics_**

~~Strikethrough~~

## Quoting

### Quoting text

> Quote

### Quoting code

```cpp
#include <iostream>
using namespace std;
int main() {
  cout << "Hello world!" << endl;
  return 0;
}
```
